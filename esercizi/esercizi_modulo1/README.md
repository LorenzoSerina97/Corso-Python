# Esercizi Modulo 1 - Python

Questa directory contiene gli esercizi del Modulo 1 del corso Python.

## Esercizi Disponibili

### 1. Laboratorio 1: Generatore di Identikit

**File:** `identikit.py`

**Obiettivo:** Creare uno script che:

- Definisca variabili per: `nome`, `anno_nascita`, `città`
- Calcoli l'età attuale sottraendo l'anno di nascita all'anno corrente
- Usi una **f-string** per stampare una frase completa

**Bonus:** Usa l'operatore `%` per stampare `True` se l'età è un numero pari, `False` se dispari.

**Esecuzione:**

```bash
python identikit.py
```

---

### 2. Laboratorio 2: Il Tuo Primo Notebook

**File:** `primo_notebook.ipynb`

**Obiettivo:** Creare un notebook interattivo con:

1. **Cella Markdown:** Titolo del notebook e breve descrizione
2. **Cella Codice #1:** Calcola l'area di un rettangolo (larghezza × altezza)
3. **Cella Markdown:** Spiega la formula usata
4. **Cella Codice #2:** Crea una lista di numeri e calcola la somma
5. **Cella Codice #3:** Usa un ciclo `for` per stampare i numeri pari da 0 a 10

**Esecuzione:**

- Apri con VS Code (con estensione Jupyter)
- Oppure carica su Google Colab

---

### 3. Progetto Cross-Modulo: Rubrica Contatti (Parte 1)

**File:** `rubrica_contatti.py`

**Obiettivo:** Definire le variabili per UN contatto e stampare le informazioni in modo formattato.

**Esecuzione:**

```bash
python rubrica_contatti.py
```

## Come Usare Questi Esercizi

1. **Prova prima da solo:** Leggi l'obiettivo e prova a scrivere il codice
2. **Consulta le soluzioni:** Se sei bloccato, guarda i file di soluzione
3. **Sperimenta:** Modifica i valori e aggiungi funzionalità extra
4. **Debug:** Usa il debugger di VS Code per capire come funziona il codice
